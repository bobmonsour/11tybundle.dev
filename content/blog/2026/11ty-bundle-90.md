## A note from Bob:

A couple of things to note:

- May 1, 2026 was the 3rd anniversary of the 11ty Bundle, and I wrote [a post celebrating this milestone](https://bobmonsour.com/blog/three-years-in-on-the-11ty-bundle/).

- I updated the Pagefind search implementation on the 11ty Bundle site to use the latest version of Pagefind along with the styling of the results. IMO, this is a significant improvement over the previous implementation as the new version of Pagefind added some much-needed features and improvements. Check out the [release notes for version 1.5.0](https://github.com/Pagefind/pagefind/releases/tag/v1.5.0).

## Highlights

**An awesome Kickstarter.** The [Kickstarter campaign for Build Awesome Pro is back](https://www.kickstarter.com/projects/fontawesome/build-awesome-pro). While it has already reached its initial goal, there are still opportunities to support the project and unlock features with its stretch goals. As of today, there are still 23 days left in the campaign, so if you haven't already, please consider backing it.

**Let's get small.** [Kitty Giraudel](https://kittygiraudel.com) brings us a reminder that [Optimizing Images with Eleventy on Netlify](https://kittygiraudel.com/2026/05/05/optimizing-images-with-11ty-on-netlify/) is pretty powerful and performant, saving tons of build time.

**Wrangling deployments.** I wrote about [Changing how I deploy my Eleventy sites](https://bobmonsour.com/blog/changing-how-i-deploy-my-eleventy-sites/), using Cloudflare's Wrangler utility. Feels like the old FTP days, only better.

**Plugin the components.** [Michael Delaney](https://mwdelaney.me/) released [zeropoint-components](https://github.com/MWDelaney/zeropoint-components#zeropoint-reusable-components), _"A configurable BuildAwesome plugin that enables a powerful component system for building dynamic, reusable HTML components across your static site."_

**Baseline improvements.** [Cris, aka 'a pleasant view'](https://www.apleasantview.com/) updated his [Eleventy Baseline plugin](https://eleventy-plugin-baseline.netlify.app/), _"an Eleventy plugin for the project plumbing most sites need from day one. You get a working starting point without giving up control of your project."_
